## FAQ

**Do I need Node.js installed?**
Yes, Node 18+.

**Can I add my own templates?**
Yes. The templates are inline in `cli.js` — easy to customize.

**Does it work offline?**
After initial `npm install -g`, yes.

**Can I use a different database?**
Prisma supports MySQL, MongoDB, and more. Just swap the provider in `schema.prisma`.
